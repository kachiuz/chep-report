How To Download CHEP Transaction Report

FIRSTLY In order for the charts to show correct data, a transaction report needs to be downloaded with these filter settings:

  • Delivery Type → Delivery Date.
  • Date Range → User Defined.
  • From To → Select a range.

SECONDLY It is very important, that delivery type in this filter is selected as "Delivery Date" and not "Create Date"!

The "Date Range" must be selected as "User Defined" for accurate month by month data, the "From" date should be the first day of the month, and the "To" date should be the last day of the month.

THIRDLY Make sure all transactions are included in the file as it truncates file to 500 transactions. If file has been truncated, this message will appear:

FOURTHLY If this message appears you have to keep clicking "View More" button, at the bottom of transactions, until all transactions ar included in the file, and only then download it!

If you wish to test this Report, you can download a prepared excel test file, and then upload it to run the report.
